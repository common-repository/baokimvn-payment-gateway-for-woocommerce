== Description ==
Tích hợp cổng thanh toán BaoKim.vn vào Woocommerce. Chỉnh sửa tên cổng thanh toán BaoKim dễ dàng như các cổng thanh toán có sẵn của Woocommerce.


== Installation ==
1. Upload thư mục \'baokimvn-woocommerce\' vào wp-content/plugins
1. Kích hoạt plugin trong khu vực Plugins -> Installed Plugins
1. Vào Woocommerce -> Settings -> Payment Gateways và chọn BaokimVN

== Frequently Asked Questions ==
= Tôi có thể tùy chỉnh thông tin cổng thanh toán này không?
Có, dễ dàng chỉnh sửa tên và mô tả cổng thanh toán này như các cổng thanh toán có sẵn trong Woocommerce.
